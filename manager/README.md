static-site-scaffold {assets manager}
===================================

## Install

```Bash
$ npm install
```

## Usage

```Bash
$ npm start
```

or

```Bash
$ grunt
```

or

```Bash
$ grunt <cmd>
```

### * Show ```<cmd>``` details.

```Bash
$ grunt -h
```

## test ( for scaffold )

```Bash
$ npm test
```

### * In the case of Windows.

```Bash
$ npm run test_win
```
